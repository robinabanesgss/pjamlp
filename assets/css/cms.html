制作の流れ

  • 1. お問い合わせ

    まずはフォームよりお気軽にお問い合わせください。後⽇、担当者より電話にてご連絡を差し上げます。納期、機能、予算、現在感じている課題などをお書き添えいただくと、より具体的なご提案が可能です。

  • 2. お⾒積り

    お困りになっていることや課題をヒアリングし、お客様にとって最適なシステムのご提案とお⾒積りをご提⽰いたします。基本的にお⾒積り作成時点では費⽤は発⽣いたしません。

  • 3. 要件定義

    担当ディレクターが、サイトマップ、ワイヤーフレーム、要件定義書、仕様書などを作成し、システムの仕組みを確定していきます。

  • 4. システム構築（設計〜開発〜検証）

    エンジニアが設計・開発・コーディングをおこないます。開発した機能は、まず弊社テスターが細かく動作検証いたします。

  • 5. ご確認

    弊社での検証が終わりましたら、お客様にテストサイトをご確認いただきます。軽微なテキスト変更や画像差し替えなどの修正があれば対応させていただきます。

    ※⼤幅なデザイン・構成の変更の場合は別途費⽤が発⽣いたしますので予めご了承ください。

  • 6. 納品

    本番サーバーにデータをアップロードし、サイトの公開確認を以って納品となります。

FAQ

  • Webやホームページに関する知識がありません。

    ホームページ制作の知識がなくても問題ございません。ご予算内でご要望を実現可能なシステムをご提案いたします。また、サイトのデザインやサーバーのご⽤意なども弊社で対応が可能です。お困りごとや不安な点がありましたらお気軽にご相談ください。

  • 社内にシステム管理や運⽤作業ができるメンバーがいません。

    弊社ではサイトの運⽤代⾏も承っておりますので、お気軽にご相談ください。

  • 相談・依頼時に、必要なものはありますか。

    以下について事前にまとまっているとヒアリングがスムーズです。

    • 業務効率を改善したいお困りポイント
    • ほしい機能
    • 希望の納期（公開スケジュール）
    • 希望のご予算
    • 弊社へのご依頼範囲（デザイン制作、サーバー準備などCMS構築以外も含め）

  • WordPressやMovableTypeの標準機能にはない機能の追加は可能ですか。

    はい、もちろんカスタマイズや機能追加も可能です。（カスタマイズの内容で費⽤や⼯期も変動します）

  • 使用するCMSのセキュリティは⼤丈夫でしょうか。

    はい、サイトのSSL化（暗号化通信）はもちろん、ログインアカウントによって役割や権限のレベルを変更することが可能です。万が⼀に備えて、システム保守契約を結んでいただくとサイトの安定稼働監視や障害発⽣時の原因調査などを弊社で実施いたします。

  • 遠⽅からの依頼は可能ですか。

    はい、お電話やメール、リモートでのミーティングなどでやり取りをしながら制作を進めることも可能です。必要な場合は担当者がお伺いいたします。
